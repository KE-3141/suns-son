### 1. トップページ `/`

**目的**
初見訪問者（元請けゼネコン・採用候補者）が会社の全体像を瞬時に把握し、問い合わせ・各ページへの回遊を促す。

**構成セクション**

| # | セクション | 内容 |
|---|---|---|
| 1 | **ヒーロー** | キャッチコピー + 背景（施工写真スライダー）+ CTAボタン |
| 2 | **平山工業について（リード文）** | 会社の一言説明・関東全域対応・50年超の実績 |
| 3 | **3つのこだわり** | 安全・高精度・安心（現行サイトより継承） |
| 4 | **事業紹介ダイジェスト** | 鉄筋工事 / 溶接工事 の簡潔な説明 + リンク |
| 5 | **施工実績ハイライト** | 代表的な現場写真 or 対応工種アイコン一覧 |
| 6 | **会社概要スニペット** | 社名・住所・TEL・地図 |
| 7 | **求人情報バナー** | 採用訴求コピー + 求人ページへのリンク |
| 8 | **お問い合わせCTA** | 電話番号大表示 + フォームへのリンク |

**キーメッセージ案**
- 「鉄筋をつなぐだけでなく、未来をつなぐ」（現行サイトより）
- 「機械ではできない。人にしかできない。」（現行サイトより）

### 2. 平山工業の強み `/about/`

**目的**
競合との差別化ポイントを伝え、問い合わせの決め手にする。

**構成セクション**

| # | セクション | 内容 |
|---|---|---|
| 1 | **ページヘッダー** | タイトル + 導入文 |
| 2 | **強み4項目** | 各強みを見出し + 本文で展開（下記参照） |
| 3 | **CTA** | お問い合わせへの誘導 |

**強み4項目（現行サイトより）**

1. **基準を満たした施工**
   設計基準・検査基準を満たす確かなノウハウ。経験豊富な職人が在籍。

2. **安全管理の徹底**
   業務前の入念な打ち合わせ・ヒヤリハット確認・安全対策の共有。事故0を目指す。

3. **質の高い提案**
   「地域の人々がより安心できる、住みよい町づくりへ貢献する」という想い。
   依頼者の立場に立った提案。

4. **さまざまな現場に対応可能**
   小規模〜大規模・高所〜地下まで対応。幅広い工種・規模の実績。

### 3. 事業紹介 `/business/`

**目的**
具体的な業務内容を伝え、発注検討者の「依頼できるか？」という疑問を解消する。

**構成セクション**

| # | セクション | 内容 |
|---|---|---|
| 1 | **ページヘッダー** | タイトル + 導入文 |
| 2 | **鉄筋工事業** | 鉄筋とは何か・なぜ重要か・当社の対応範囲 |
| 3 | **鉄筋組立の工程** | 搬入 → ベース配筋 → 柱配筋 → 梁配筋 の流れ |
| 4 | **溶接工事** | 杭頭補強筋溶接（フレア溶接工法）の説明 |
| 5 | **施工事例写真** | 浄水場・現場風景など |
| 6 | **対応エリア** | 関東全域（地図） |
| 7 | **CTA** | お問い合わせへの誘導 |

**コンテンツポイント**
- 「鉄筋は建造物の命」という価値訴求
- 加工・運搬・組立の一貫対応を明示
- 提携工場（㈱松伸・有限会社セイコー）への言及

### 4. 技術紹介 `/technology/`

**目的**
他社にない溶接技術の専門性を示し、技術力による差別化を図る。

**構成セクション**

| # | セクション | 内容 |
|---|---|---|
| 1 | **ページヘッダー** | タイトル + 導入文（多能工・一括請負の強み） |
| 2 | **フレア溶接** | 概要・メリット（複雑形状対応・耐震性向上） |
| 3 | **杭頭筋溶接** | 平山工業独自技術 → 業界標準化した実績 |
| 4 | **J-BAR溶接（NewJ-BAR）** | 溶接欠陥抑制・信頼性の高い継手 |
| 5 | **エンクローズ溶接（CB工法）** | 狭所施工対応・セラミック裏当材使用 |
| 6 | **鋼管杭ズレ止溶接** | リング溶接による補強 |
| 7 | **CTA** | お問い合わせへの誘導 |

**コンテンツポイント**
- 杭頭筋の半自動溶接化は業界先駆けの独自技術として強調
- 各工法に施工写真・図解を添えると効果的（要素材確認）

**目的**
品質管理・安全管理への取り組みを示し、発注者の「任せて大丈夫か」という不安を解消する。

**構成セクション**

| # | セクション | 内容 |

| 5 | **品質への追求** | 日々の点検・工程ごとの自主検査・社員の責任感 |

- 「全現場における事故0」という数値目標を明示

### 7. 求人情報 `/recruitment/`

**目的**
鉄筋工・溶接工の採用を促進する。経験者・未経験者どちらにも響くページにする。

**構成セクション**

| # | セクション | 内容 |
|---|---|---|
| 1 | **ページヘッダー** | タイトル + 採用コピー |
| 2 | **採用メッセージ** | 経営理念「良い製品は良い人がつくる」 + ご挨拶文 |
| 3 | **会社の魅力3つ** | アットホーム・教育制度・働きやすい環境（単身寮・独立支援） |
| 4 | **求人票** | 鉄筋工・溶接工の募集要項テーブル |
| 5 | **求める人材** | 3つの条件（マナー・協調性・責任感） |
| 6 | **選考フロー** | 問い合わせ → 面接 → 合否通知（1週間）→ 勤務開始 |
| 7 | **CTA** | 電話・フォームへの誘導 |

**募集要項テーブル**

| 項目 | 鉄筋工 | 溶接工 |
|---|---|---|
| 業務内容 | 現場での鉄筋組立 / 加工場での鉄筋材加工 | 鉄筋材への溶接 / 基礎工事での杭頭筋補強溶接 |

| 勤務地 | 東京都ほか関東全域 | 同左 |
| 勤務時間 | 8:00〜17:00（現場により異なる） | 同左 |
| 休日 | 日祝・夏季・年末年始・会社規定休日 | 同左 |
| 給与 | [要確認] | [要確認] |
| 待遇 | 残業手当・単身用寮・独立支援・資格支援 | 同左 |
